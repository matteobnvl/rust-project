# Rust Project

## Démarrage rapide

### Prérequis
- Rust (stable) et Cargo installés → [https://rustup.rs](https://rustup.rs)
- Un terminal compatible avec le rendu **ANSI** (couleurs et symboles).

Build:
```bash
cargo build
```

Run:
```bash
cargo run
```

Run in release mode (faster):
```bash
cargo run --release
```
